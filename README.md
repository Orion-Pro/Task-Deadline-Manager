# Task-Deadline-Manager

A simple project in concept, but with plenty of room for improvement, practice, and learning experiences.
Working with java's built in LocalDate and LocalDateTime data types and methods seemed rudimentary and useful to learn.
This project also allows me to practice and 'play arround' with comparators, sorting functions, data structures (I'll probably rework the data storage system).
Furthermore, one of my objectives is to load data from files, and write local data to files as well.
